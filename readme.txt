== Description ==

LianaMailer Contact Form 7 is an integration plugin for a [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) plugin which integrates LianaMailer into Contact Form 7.

= Contribute and translate =

LianaMailer Contact Form 7 plugin is developed and supported by Liana Technologies Oy. Join our crew at GitHub or contact using other means.

== Frequently Asked Questions ==

= Where can I find LianaMailer documentation and user guides? =

For help setting up and configuring Liana Mailer, please refer to [LianaMailer support site](https://support.lianatech.com/lianamailer.html).

= Where can I report bugs? =

Report bugs on our [GitHub repository](https://github.com/LianaTechnologies/lianamailer-contact-form-7/issues).

You can also notify us via our support channels.
